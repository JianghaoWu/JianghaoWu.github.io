Residential safety has become a topic of great concern, with 47.9% of recent burglary cases in Chengdu involving entry through windows. Incidents like accidental falls from heights, objects falling from upper floors, and obstructed fire escape routes due to anti-theft window installations have resulted in injuries and fatalities.

In response, this project aims to design an aesthetically pleasing, secure, and practical smart invisible security net. It consists of lifting mechanisms, lateral shifting structures, self-locking mechanisms, and manual opening mechanisms. This system not only provides basic anti-theft functionality but also incorporates automatic fire alarm features and intelligent auto-open/close functions. In case of a fire or when users need it, the system can swiftly and automatically open. Additionally, in case of any malfunctions, it can be manually opened quickly. This innovation ensures both safety and comfort in home living environments.
